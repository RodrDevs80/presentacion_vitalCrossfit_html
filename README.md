# VitalCrossfit - Plataforma E-commerce Fitness

Este código es una presentación web interactiva sobre **VitalCrossfit**, una plataforma de e-commerce especializada en fitness y crossfit.

## Descripción General

**Estructura**: Presentación de diapositivas con 11 slides que explican el proyecto

### Contenido Principal

- Presentación del proyecto VitalCrossfit
- Arquitectura del sistema (React + Node.js + MySQL)
- Tecnologías utilizadas (frontend y backend)
- Características principales del e-commerce
- Sistema de seguridad implementado
- Instrucciones de instalación
- Próximas funcionalidades planeadas

## Características Técnicas

- **Diseño**: Responsive con gradientes morados/azules
- **Navegación**: Entre slides con botones y teclado (flechas)
- **Indicadores**: Contador de diapositivas
- **Animaciones**: Efectos suaves (fadeIn, slideUp)
- **Interactividad**: Tarjetas con efectos hover
- **Optimización**: Código adaptado para diferentes tamaños de pantalla

## Funcionalidades

- ✅ Navegación secuencial entre slides
- ✅ Indicador de progreso (slide actual/total)
- ✅ Diseño visual atractivo con tarjetas y badges tecnológicos
- ✅ Completamente funcional en navegadores modernos

## Propósito

Esencialmente, este proyecto es un **pitch de presentación** en formato web para mostrar las capacidades del proyecto VitalCrossfit, destacando su arquitectura técnica, características principales y visión de futuro.
